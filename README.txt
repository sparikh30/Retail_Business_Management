1.	Open project through new beans IDE through File -> Open project.

2.	Change Username and password in NewJFrame.java file on line number 123. Set your oracle username at uname.equals(XXX) and set oracle password at paswd.equals(XXX)

3.	Project will be open in left side Projects window. Right click and run the project.
